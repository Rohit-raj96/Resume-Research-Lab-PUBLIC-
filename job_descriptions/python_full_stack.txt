Job Description: Junior Python Full Stack Developer
Role Overview
We are looking for a motivated and talented Python Full Stack Developer (Fresher) to join our engineering team. You will be responsible for developing and maintaining both the front-end and back-end of web applications. You will work closely with senior developers to build scalable software solutions, write clean code, and participate in the full software development life cycle (SDLC).

Key Responsibilities
Back-End Development: Write robust, testable, and efficient code using Python and frameworks like Django or Flask.

Front-End Development: Build responsive and user-friendly web interfaces using HTML5, CSS3, and JavaScript frameworks (React or Angular).

Database Management: Design and optimize database schemas using PostgreSQL, MySQL, or MongoDB.

API Integration: Create and consume RESTful APIs to connect the front-end with back-end services.

Testing & Debugging: Write unit tests and debug code to ensure high-quality delivery.

Collaboration: Participate in code reviews, daily stand-ups, and contribute to technical documentation.
